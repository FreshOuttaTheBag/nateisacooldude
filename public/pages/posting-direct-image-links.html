How to Natepost:

A quick guide

1.) Stroll on down to your favorite image hosting website

For this example we will use imgur.com

2.) Post or find an image that you like

I've grown really fond of this GIF of a hamster running

3.) Right-Click on the image and select "copy image address"

4.) Paste the link you just copied into the URL field on the new post page
